## File Auto Transfer

一个用于在两个文件夹之间进行 **增量复制 / 自动转移文件** 的图形界面小工具。

---

## 🧰 功能介绍

- 支持复制来源 ➜ 目标目录文件  
- **只复制新文件** 或 **变更过的文件**
- 两种校验模式：
  - `fast` - 比较文件大小 + 修改时间（速度快）
  - `hash` - 使用 SHA256 校验（更严谨但更慢）
- 带有图形化界面
- 文件过滤支持
- 单文件进度条显示
- 暂停、继续、停止控制
- 支持执行完成后可选**删除来源文件**
- 自动记录日志到 `copy_log.txt`

---

## 🖥️ 运行环境

- Windows 10 / 11
- 已安装 Python 3.x（建议 Python 3.8+）

---

## 🚀 使用方法

1. 双击 `copy_files_tool.py`（如果已把 `.py` 关联到 Python），或者在终端中运行：

```bash
python copy_files_tool.py
```

---

2. 在界面中设置：

- 来源目录（默认值可在配置文件中修改）
- 目标目录
- 检测模式： `fast` 或 `hash`
- 是否勾选「复制完成后删除来源文件」

---

3. 点击【开始复制】，等待任务完成即可。
